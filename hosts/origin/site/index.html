bougie.

A uv for PHP.

A Composer-compatible package manager, PHP toolchain manager, and dev server — in one fast Rust binary. Install PHP and extensions, resolve and install dependencies, and bring up your services with a single command.

$curl -LsSf https://bougie.tools/install.sh | sh

Windows: irm https://bougie.tools/install.ps1 | iex

Fast by construction

A native, pubgrub-based Composer resolver and an autoloader that's byte-for-byte identical to Composer's — written in Rust, parallel throughout. composer install, in seconds.

The whole toolchain

Installs and pins PHP versions and extensions, runs Composer, and supervises dev services — MariaDB, Redis, OpenSearch, RabbitMQ — so a project comes up the same way everywhere.

Drop-in compatible

Reads your composer.json and composer.lock and produces the same vendor/. No new manifest, no workflow change — adopt it today.

One command to run

bougie sync creates the lockfile if needed, installs the matching PHP + extensions, and materializes vendor/. bgx runs a tool straight from a Composer package.

Built to make Magento approachable

Magento has the heaviest dev setup in PHP. bougie takes it from "follow the 20-step wiki" to "one command to a running store": the right PHP, every required extension, the full service stack, and a hundreds-of-package install that finishes in seconds.
